# Debug Panel

A lightweight in-game debug panel for Unity developers, packaged for installation through Unity Package Manager.

## Install

Use **Add package from git URL** in Unity Package Manager:

```text
https://github.com/taycpu/TestConsole.git?path=/Packages/com.taycpu.debug-panel
```

- Package README: [`Packages/com.taycpu.debug-panel/README.md`](Packages/com.taycpu.debug-panel/README.md)
- Full documentation: [`Packages/com.taycpu.debug-panel/documentation.md`](Packages/com.taycpu.debug-panel/documentation.md)
